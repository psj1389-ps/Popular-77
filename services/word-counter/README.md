# 글자수 및 변환기 (Word Counter & Text Converter)

실시간 텍스트 분석 및 변환 도구입니다.

## 🚀 기능

### 📊 텍스트 분석
- **글자수 계산**: 공백 포함/제외 글자수
- **단어수 계산**: 정확한 단어 개수 측정
- **문장수 계산**: 문장 구분 및 개수 측정
- **문단수 계산**: 문단 구분 및 개수 측정
- **읽기 시간**: 평균 읽기 속도 기반 예상 시간

### 🔄 텍스트 변환
- **대문자 변환**: 모든 텍스트를 대문자로
- **소문자 변환**: 모든 텍스트를 소문자로
- **각 단어 첫 글자 대문자**: 단어별 첫 글자만 대문자
- **공백 제거**: 모든 공백 제거
- **중복 공백 정리**: 여러 공백을 하나로 통합

### ⚙️ 고급 기능
- **문자 제한**: 3,000 / 5,000 / 10,000자 제한 선택
- **영어 텍스트 감지**: 영어 전용 기능 자동 활성화
- **실시간 분석**: 입력과 동시에 즉시 분석
- **반응형 디자인**: 모든 디바이스에서 최적화

## 🛠️ 기술 스택

- **Frontend**: Next.js 14, React 18
- **상태 관리**: MobX State Tree
- **스타일링**: Tailwind CSS
- **언어**: JavaScript (ES6+)

## 🎯 사용법

1. **텍스트 입력**: 왼쪽 텍스트 영역에 분석할 텍스트를 입력
2. **실시간 분석**: 오른쪽에서 실시간으로 분석 결과 확인
3. **문자 제한 설정**: 우상단 버튼으로 문자 제한 변경
4. **텍스트 변환**: 하단 변환 도구로 텍스트 형태 변경
5. **설정 조정**: 우상단 설정 패널에서 표시 옵션 조정
